= elin-coc-source

https://github.com/neoclide/coc.nvim[coc.nvim] source for https://github.com/liquidz/elin[liquidz/elin].

== Installation

.vim-plug
[source,vim]
----
Plug 'liquidz/elin-coc-source'
----

.dein
[source,vim]
----
call dein#add('liquidz/elin-coc-source')
----
